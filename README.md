

## Highlighting

Install the Highlight plugin by Fabio Spampinato.

Configure it with (included in workspace already):

```
    "highlight.regexes": {
        "({)(.*)(})": {
            "filterLanguageRegex": "markdown", 
            "decorations": [
                {},
                {
                    "color": "#569CD6",
                },
                {}
            ],
        }
    },
```
